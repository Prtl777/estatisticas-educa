Estatística

O que é Estatística?

A Estatística é a ciência que se dedica à coleta, organização, análise, interpretação e apresentação de dados. Ela nos permite transformar dados brutos em informações úteis para a tomada de decisões.

Divisões da Estatística

A Estatística pode ser dividida em duas grandes áreas:

1. Estatística Descritiva

Organiza e descreve os dados através de tabelas, gráficos e medidas resumo (média, mediana, moda, desvio padrão, etc.). Seu objetivo é resumir e apresentar as características principais de um conjunto de dados.

2. Estatística Inferencial

Utiliza os dados de uma amostra para fazer inferências e tirar conclusões sobre uma população maior. Envolve testes de hipóteses, intervalos de confiança e previsões.

Importância da Estatística

  • Auxilia na tomada de decisões baseadas em dados
  • Permite identificar padrões e tendências
  • Facilita a comunicação de informações complexas
  • É fundamental em pesquisas científicas
  • Aplicável em diversas áreas: economia, saúde, educação, etc.

Frequência Absoluta (fi)

É o número de vezes que cada valor aparece no conjunto de dados.

Frequência Relativa (fr)

É a razão entre a frequência absoluta e o total de observações. Pode ser expressa em decimal ou porcentagem.

fr = fi / n

Frequência Acumulada (Fac)

É a soma das frequências absolutas até determinado valor.

Criação de Tabelas de Frequência

Passo a Passo para Criar uma Tabela de Frequência

1

Coletar os Dados

Reúna todos os dados que serão analisados. Exemplo: notas de uma turma.

2

Organizar em Rol

Ordene os dados em ordem crescente ou decrescente para facilitar a contagem.

3

Determinar as Classes (se necessário)

Para dados contínuos ou com muitos valores diferentes, agrupe-os em intervalos (classes).

Número de classes ≈ √n (onde n é o total de dados)

4

Calcular a Amplitude

Amplitude = Maior valor - Menor valor

Amplitude de classe = Amplitude total / Número de classes

5

Contar as Frequências

Conte quantas vezes cada valor ou classe aparece nos dados.

6

Calcular Frequências Relativas e Acumuladas

Complete a tabela com as frequências relativas (fi/n) e acumuladas (soma progressiva).

Tipos de Gráficos

Principais Tipos de Gráficos Estatísticos

1. Gráfico de Barras

Utilizado para comparar valores entre diferentes categorias. As barras podem ser horizontais ou verticais.

Quando usar: Comparação de categorias distintas (ex: vendas por região, preferências, etc.)

2. Gráfico de Linhas

Ideal para mostrar tendências e mudanças ao longo do tempo.

Quando usar: Dados temporais, evolução de valores, séries históricas

3. Gráfico de Setores (Pizza)

Mostra a proporção de cada categoria em relação ao todo. Cada fatia representa uma porcentagem do total.

Quando usar: Mostrar partes de um todo, composição percentual

4. Gráfico de Colunas

Similar ao gráfico de barras, mas com colunas verticais adjacentes (sem espaços). Muito usado para mostrar frequências de dados agrupados em classes.

Quando usar: Distribuição de frequências, dados agrupados em intervalos

5. Gráfico de Dispersão

Mostra a relação entre duas variáveis através de pontos em um plano cartesiano.

Quando usar: Analisar correlação entre variáveis, identificar padrões

Dicas para Criar Bons Gráficos

  • Escolha o tipo de gráfico adequado para seus dados
  • Use títulos claros e descritivos
  • Identifique os eixos com suas respectivas unidades
  • Use cores contrastantes e legíveis
  • Evite poluição visual - mantenha o gráfico simples
  • Inclua legendas quando necessário

Histogramas

O que é um Histograma?

O histograma é um gráfico de barras verticais que representa a distribuição de frequências de dados quantitativos agrupados em classes ou intervalos. Diferente do gráfico de barras comum, no histograma as barras são adjacentes (coladas), sem espaços entre elas.

Características do Histograma

  • Barras verticais adjacentes (sem espaços)
  • Eixo horizontal: classes ou intervalos de valores
  • Eixo vertical: frequência (absoluta ou relativa)
  • A área de cada barra é proporcional à frequência
  • Usado para dados contínuos ou agrupados em classes

Como Construir um Histograma

1

Organize os dados em classes

Determine o número de classes e a amplitude de cada uma.

2

Calcule as frequências

Conte quantos dados pertencem a cada classe.

3

Desenhe os eixos

Eixo horizontal com as classes e eixo vertical com as frequências.

4

Construa as barras

Desenhe barras adjacentes com altura proporcional à frequência de cada classe.

Exemplo de Histograma

Distribuição de alturas (em cm) de 30 alunos:

Classe (altura em cm) Frequência
150 ├ 155 3
155 ├ 160 7
160 ├ 165 12
165 ├ 170 6
170 ├ 175 2

Interpretação do Histograma

O histograma permite visualizar:

  • Distribuição dos dados: Como os valores estão espalhados
  • Tendência central: Onde se concentra a maioria dos dados
  • Dispersão: Quão espalhados estão os valores
  • Assimetria: Se a distribuição é simétrica ou assimétrica
  • Valores atípicos: Dados que fogem do padrão

Vantagens do Histograma

  • Visualização clara da distribuição de frequências
  • Fácil identificação de padrões e tendências
  • Útil para grandes conjuntos de dados
  • Permite comparação visual entre diferentes distribuições
  • Auxilia na tomada de decisões baseadas em dados
